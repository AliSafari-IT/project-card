## 🏃‍♂️ How to Run

### Quick Start

```bash
# From the project-card directory
npm run demo
```

### Manual Steps

```bash
# Build the package
npm run build

# Install demo dependencies
cd demo
npm install

# Start the demo
npm run dev
```

### Using Scripts

```bash
# Linux/Mac
./demo-setup.sh

# Windows
demo-setup.bat
```
